# QuickBite – React Food Ordering Template

QuickBite is a modern food ordering UI template built with React and Tailwind CSS.

Designed for restaurants, food delivery startups, and online menu websites.

---

## Features

- Modern responsive design
- Cart system with quantity control
- Checkout UI flow
- Order success page
- Meal filtering and sorting
- Category browsing
- WhatsApp checkout integration
- Light / Dark mode
- Framer Motion animations
- Mobile-first UI

---

## Tech Stack

React  
Tailwind CSS  
Framer Motion  
React Router  

---

## Installation

1. Clone the repository


git clone https://github.com/gracewebpro/quickbite


2. Install dependencies


npm install


3. Start development server


npm start


---

## Build for Production


npm run build


---

## Editing Meals

Meals are located in:


src/data/mealsData.js


Example:


{
id: 1,
name: "Chicken Burger",
price: 4500,
category: "fastfood",
image: "image-url"
}


---

## Editing Categories

Edit:


src/data/categories.js


---

## Changing Colors

Edit:


tailwind.config.js


Example:


primary: "#FF6B35"
secondary: "#FF9F1C"
